# Memo

## PlantUML

### Requirements

- Graphviz 2.44
    ```console
    $ curl -LO https://www2.graphviz.org/Packages/stable/portable_source/graphviz-2.44.1.tar.gz
    $ tar xvzf graphviz-2.44.1.tar.gz
    $ cd graphviz-2.44.1/
    $ ./configure
    $ make
    $ sudo make install
    ```
- PlantUML
    ```console
    $ sudo apt -y install plantuml
    ```
- IPA fonts
    ```
    $ sudo apt -y install fonts-ipafont fonts-ipaexfont
    ```
